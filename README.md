## :wrench:	Installation

1.  **Clone this repo**

  ```shell
  https://github.com/Luan-Nagel/intibiome-web-page.git
  ```

## :rocket: Run the project

2.  **Navigate to the project directory**

    Use the Gatsby CLI to create a new site, specifying the minimal starter.

    ```shell
    cd intibiome-web-page/
    ```

3.  **Start developing**

    ```shell
    npm run develop
    ```
